Weather App

Weather App: A clean, responsive web app that delivers real-time weather updates. I built it to strengthen my skills in working with live data and third-party APIs while keeping the interface simple and intuitive.

What I Learned

API Integration: Practiced making asynchronous requests to a weather API, parsing JSON, and updating the UI on the fly.

Tech Stack

  • HTML
  • Tailwind CSS
  • JavaScript

The Creative Process

I focused on clarity and usability: a minimalist color scheme, a single search field, and clear data presentation. The goal was a seamless user experience even for quick, one-handed checks on a mobile device.
