Netflix Clone

Netflix Clone: This project is a full front-end recreation of the Netflix interface. My goal was to capture the feel of the original site while improving my ability to design complex, responsive layouts and interactive features.

What I Learned

Advanced CSS & Layouts: I worked extensively with Flexbox and Grid to build a fluid multi-column design. Careful spacing and alignment helped match the authentic Netflix look across screens large and small.

Dynamic UI with JavaScript: I built horizontal scrolling carousels and hover effects, using DOM manipulation to create smooth transitions and interactive elements that react instantly to user input.

Tech Stack

  • HTML
  • Tailwind CSS
  • JavaScript
